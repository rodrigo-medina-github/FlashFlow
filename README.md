# FlashFlow – Aplicação de Flashcards

## Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio prático da minha pós-graduação.
O objetivo foi criar uma aplicação simples de flashcards utilizando HTML, CSS e JavaScript.

A aplicação exibe perguntas em um card e revela a resposta quando o usuário clica nele.
Esse método utiliza repetição ativa para auxiliar no processo de aprendizado.

### Funcionalidades

A aplicação mostra um flashcard com uma pergunta e permite revelar a resposta ao clicar no card.
Um botão permite avançar para o próximo card.

Também é exibido um contador indicando a posição atual do card.

### Tecnologias

HTML foi utilizado para a estrutura da página.
CSS foi utilizado para estilização e layout.

JavaScript foi utilizado para controlar a lógica da aplicação e manipulação do DOM.

### Objetivo

O objetivo deste desafio foi praticar manipulação do DOM e tratamento de eventos.
Também buscou reforçar conceitos de renderização dinâmica e organização de interface.
